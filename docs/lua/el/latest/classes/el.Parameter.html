Methods

Get the current value.

Returns:

    number The currrent value
Parameter:set (value)
Set the current value.

Parameters:

  • value number The new value to set

Handlers

Parameter:valuechanged ()
Value changed handler. Implement this to handle when the value changes. If you call Parameter:set from inside this callback, be careful to not create an infinte callback loop.

Usage:

    -- The first argument, self, will be the Parameter object
param.valuechanged = function (self)
    local value = self:get()
    print (value)
end
