Class kv.MidiBuffer

A MIDI buffer.

Designed for real time performance, therefore does virtually no type checking in method calls.

MidiBuffer:swap (other)
Exchanges the contents of this buffer with another one. This is a quick operation, because no memory allocating or copying is done, it just swaps the internal state of the two buffers.

Parameters:

  • other kv.MidiBuffer Buffer to swap with
MidiBuffer:insert (data, frame)
Insert some MIDI in the buffer.

Parameters:

  • data int Packed integer data
  • frame int Sample index to insert at
MidiBuffer:addbytes (bytes, size, frame)
Insert some bytes into the buffer. The kv.ByteArray passed in should contain a complete MIDI message of any type.

Parameters:

  • bytes kv.ByteArray The bytes to add
  • size int Max number of bytes to add
  • frame int Sample index to insert at
MidiBuffer:events ()
Iterate over MIDI data. Iterate over midi data in this buffer

Returns:

    Event data iterator

Usage:

    -- @data    Raw byte array
-- @size    Size in bytes
-- @frame   Audio frame index in buffer
for data, size, frame in buffer:events() do
    -- do something with midi data
end

end
