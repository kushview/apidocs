Attributes

TextButton.togglestate
The button's toggle state. Setting this property will notify listeners. If you need to change the toggle state and NOT notify, use the settogglestate method instead.
TextButton.text
Displayed text.

Methods

TextButton:settogglestate (state)
Change the button's toggle state. Note this variation will send notifications to listeners.

Parameters:

  • state bool The new toggle state on or off
TextButton:settogglestate (state, notify)
Change the button's toggle state.

Parameters:

  • state bool The new toggle state on or off
  • notify bool If true sends notification to listeners

On clicked handler. Executed when the button is clicked by the user.
